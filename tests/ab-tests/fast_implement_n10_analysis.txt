================================================================================
KEY FINDING: BIMODAL BASELINE
================================================================================

The without-pruner side has bimodal behavior:
  - "Efficient" rounds (R3, R5, R6, R8, R9): 9-10 tool calls, $0.08-0.10
  - "Exploratory" rounds (R0, R1, R2, R4, R7): 17-25 tool calls, $0.12-0.14

With pruner is more consistent: 6-12 tool calls, $0.08-0.13

When the baseline is already efficient (9-10 tools), pruner adds overhead
for no benefit (+4% to +157% cost). When the baseline explores heavily
(17-25 tools), pruner saves significantly (-7% to -37% cost).

This explains the high variance: the benefit depends on whether the
baseline would have been efficient anyway. Median (-14% cost, -33% tools)
is more representative than mean (+2% cost).
